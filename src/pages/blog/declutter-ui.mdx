## An interface is there to be used

So why am I ranting to you about my beautiful but frustrating home devices?

Because I want to make the point that an interface is there to be used not looked at. Our users are there to complete a task. As designers our job is to make that as easy and efficient for them as possible.

So coming back to the earlier example, it's not "clutter" if the user needs the hints in order to complete the form correctly.

Good design is content-driven not aesthetics-driven.

Respect people's time. Create useful things.
